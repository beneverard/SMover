SMover Demo

SMover (Social Media hover) is a jQuery plugin that hides social media icons away, whilst allowing super smooth and easy access to them when required. Whilst the main use for this plugin is for social media icons, it has other uses too.

Examples

Basic Use

SMover in it's most basic form can be used out of the box for social media links:

Share Page
 

Usage

JavaScript

				$(document).ready(function() {
					$('#share-page').smover();
				});

Other Uses

SMover is actually so basic that it can be used for many purposed, in this instance we're hiding away icons that signify actions on a document.

Notice the fillerText has changed to ' to ' and the mouseoutDelay has increased to 3000 milliseconds.

Send Document
 

Usage

JavaScript

				$(document).ready(function() {
					$('#send-document').smover({
						fillerText    : ' to ',
						mouseoutDelay : 3000
					});
				});
